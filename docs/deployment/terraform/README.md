# Terraform

This directory contains examples using [Terraform](https://www.terraform.io) for infrastructure as code:

1. [**GCE**](gce/README.md): deploy the application on [Compute Engine](https://cloud.google.com/compute/) with a WAR file, as described in [this guide](../gcp/gce/README.md).
2. [**GKE**](gke/README.md): deploy the application on [Kubernetes Engine](https://cloud.google.com/kubernetes-engine/) as described in [this guide](../gcp/gke/README.md).
3. [**GKE Cluster**](gke-cluster/README.md): create a [Kubernetes Engine](https://cloud.google.com/kubernetes-engine/) cluster for use with the previous example.
4. [**Cloud Run**](cloudrun/README.md): deploy the application on [Cloud Run](https://cloud.google.com/run) with a container image, as described in [this guide](../gcp/cloudrun/README.md).
